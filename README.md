# 봉인된 자

AI 생성 영상과 직접 편집을 활용해 제작한 인터랙티브 스토리 게임입니다.  
본 프로젝트는 개인 창작물 및 포트폴리오 공개를 목적으로 제작되었습니다.

## 작품 소개

「봉인된 하나」는 선택지에 따라 이야기가 분기되는 웹 기반 인터랙티브 스토리입니다.  
HTML, CSS, JavaScript를 사용해 제작했으며, 영상 배경과 텍스트 연출을 중심으로 구성되었습니다.

## 제작 방식

- 코드: HTML, CSS, JavaScript
- 영상: 상업적 이용이 가능한 AI 영상 생성 도구를 활용한 뒤 After Effects로 직접 편집
- 그래픽 에셋: 직접 제작 또는 프로젝트용으로 생성한 에셋 사용
- 음향: 

## AI 활용 고지

본 작품의 일부 이미지 및 영상은 AI 도구를 활용해 생성한 뒤, 제작자가 직접 편집하여 완성했습니다.

## 저작권 및 이용 안내

본 저장소는 포트폴리오 검토와 작품 공개를 위해 공개되어 있습니다.  
그러나 저장소가 공개되어 있더라도, 아래 항목의 무단 사용은 허용하지 않습니다.

- 스토리와 문장 복제
- 영상 파일 다운로드 후 재업로드
- 이미지 및 그래픽 에셋 재사용
- 게임 구성과 연출을 그대로 복제한 2차 배포
- 상업적 이용을 위한 무단 사용
- AI 학습

본 프로젝트의 스토리, 문장, 영상, 이미지, 그래픽 에셋, 연출 구성은 제작자의 창작물입니다.  
별도의 허가 없이 재사용하거나 배포할 수 없습니다.

## Fonts

본 프로젝트에는 무료 사용 및 웹 임베딩이 허용된 폰트를 사용했습니다.  
각 폰트의 저작권은 해당 제작자에게 있습니다.
